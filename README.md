# Power System Fault Detection and Classification

IBM SkillsBuild AICTE 2026 Internship Project — Problem Statement No. 41

## Overview
A machine learning model built using IBM watsonx.ai Studio AutoAI to classify 
power system faults (Line Breakage, Overheating, Transformer Failure) based on 
electrical and environmental measurements.

## Contents
- `fault_data.csv` – Kaggle dataset used for training
- Problem Statement PDF
- Project PPT
- Notebook.ipynb – Python notebook exported from IBM watsonx.ai AutoAI

## Technology
IBM Cloud Lite, Watson Studio, AutoAI, watsonx.ai Runtime

## Result
Best model: Snap Logistic Regression — Accuracy: 39.3%
